O Stronie

Projekt jest stroną typu OnePage z osobną pod stroną dla opisu projektu oraz zainteresowań. Jest to wizytowka prezentująca opis, informacje, zdjęcia, dane kontaktowe, możliwość zapisania się do newsletera z podaniem adresu e-mail oraz kontaktu podając imie, nazwisko, teamt, nr. telefonu i wiadomość ktorą chcemy przesłać do osoby którą dotyczy. Strona zostałą napisana z wykorzystaniem HTML, CSS, JavaScriptu, PHP oraz SQL. Poniżej postaram sie opisać w skrócie użyte technologie oraz odwołania do nich. Całe repozytorium projektu jest umieszczone na GitHub-ie.

Strona składa się z siedmiu sekcji mających na celu intuicyjne poruszanie się na stronie oraz odpowiednie przedstawienie zamieszczonych treści. pasek nawigazji posiadamy cały czas na góże strony dzięki czemu osoba przeglądająca cały czas ma informacje w której częśći strony się znajduję oraz ma możliwość swobodnego poruszania się po stronie. do tego w lewym rogu mamy możliwość przejścia na góre strony.

Opis sekcji



*Sekcja pierwsza
Baner z zawartym mottem.

*Sekcja druga
Opis osoby której dotyczy strona.

*Sekcja trzecia
Kategorie na podstawie których możemy przejść do podstron, np. wpisó na dany temat.

*Sekcja Czwarta
Galeria zdjęć, układająca zdjęcia losowo przy każdym odwiedzeniu strony.

*Sekcja piąta
Skrócony opis projektu z możliwością przejścia do obecnej strony.

*Sekcja szósta
Sekcja kontaktu z autorem strony, przekazywana jest wiadomość zawierająca teamt, treść, dane kontaktowe w postaci nr. telefonu, adresu e-mail oraz imienia/pseudonimu.

*Sekcja siódma
Stopka strony z informacjiami kontaktowymi oraz z możliwością zapisania się do newslettera.

Uważam że strona spełnia założenia projektowe, ponieważ wykorzystuje technologie wymienione w wymaganiach projektowych. Strona korzysta z różnych stylów CSS oraz JavaScriptu. Które nadają stronie atrakcyjny wygląd oraz zapewniają przyjemna dla urzytkownika efekty wizualne np. różnego rodzaju przejścia, animacje, podświetlanie. Wykorzystuje PHP do backendu strony umieszczonego na hostingu odpowiedzialnego za połączenie do bazy danych oraz wprowadzeni danych do bazy w sekcji "Kontakt" gdzie przesyłana jest treść wiadomośći wraz z danymi kontaktowymi oraz sekcji "Newsletter" do przesłąnia adresu mailowego który możemy wykorzystać do poinformowania urzytkowników, gdy dodamy nowe informacje/wpisy np. zainteresowania.

Technologie

Opis wykorzystanych technologi

Aos

Aos jest to zbiór animacji okien i nie tylko. Idea AOS jest prosta obserwuj wszystkie elementy i ich pozycje w oparciu o ustawienia, które im podasz. CSS obsługuje każdy aspekt animacji. Pniżej link do projektu AOS.
AOS

bootstrap.min

Bootstrap to darmowy i otwarty framework CSS ukierunkowany na responsywne, front-endowe tworzenie stron internetowych.
Bootstrap

font-awesome.min

Font Awesome zapewnia skalowalne ikony wektorowe, które można natychmiast dostosować - rozmiar, kolor, cień i wszystko, co można zrobić za pomocą CSS.
Font Awesome

LineIcons

Ręcznie wykonane ikony linii dla nowoczesnych interfejsów użytkownika projektów internetowych, Android, iOS i aplikacji komputerowych. Kompletny, ale najbardziej uproszczony bezpłatny pakiet ikon dostępny w sieci!
LineIcons

Slick

Suwak lub pasek toru to graficzny element sterujący, za pomocą którego użytkownik może ustawić wartość, przesuwając wskaźnik, zwykle poziomo.
Slider

JavaScript

Opis wykorzystanych skryptów JS

Modernizr

Bezpłatna biblioteka napisana w języku JavaScript, stworzona do wykrywania dostępności funkcji HTML5 oraz CSS3 w przeglądarkach użytkowników, co pozwala deweloperowi na alternatywne użycie starszych, szerzej kompatybilnych funkcji i składni.

jQuery

Lekka biblioteka programistyczna dla języka JavaScript, ułatwiająca korzystanie z JavaScriptu (w tym manipulację drzewem DOM). Kosztem niewielkiego spadku wydajności w stosunku do profesjonalnie napisanego kodu w niewspomaganym JavaScripcie pozwala osiągnąć interesujące efekty animacji, dodać dynamiczne zmiany strony, wykonać zapytania AJAX. Większość wtyczek i skryptów opartych na jQuery działa na stronach nie wymagając zmian w kodzie HTML (np. zamienia klasyczne galerie złożone z miniatur linkujących do obrazków w dynamiczną galerię).

wow.min

Zestaw ciekawych i miłych dla oka efektów związanych z przewijaniem strony internetowej. Narzędzie to korzysta z biblioteki animate.css, wykorzystując jej całe dobrodziejstwo.

SQL

SQL został wykorzystany do utworzenia bazy danych oraz w plikach php do przekazania danych, poniżej zostały przedstawione wykorzystane zapytania z któtką informacją do nich.


Utworzenie tabeli Conntact oraz Email.
CREATE TABLE `Conntact` (
`id` int(11) NOT NULL,
`Name` varchar(155) COLLATE utf8_polish_ci NOT NULL,
`Email` varchar(155) COLLATE utf8_polish_ci NOT NULL,
`Phone` int(11) NOT NULL,
`Subject` varchar(155) COLLATE utf8_polish_ci NOT NULL,
`Message` varchar(155) COLLATE utf8_polish_ci NOT NULL
) DEFAULT CHARSET=utf8 COLLATE=utf8_polish_ci;

CREATE TABLE `Email` (
`id` int(11) NOT NULL,
`Email` varchar(155) COLLATE utf8_polish_ci NOT NULL
) DEFAULT CHARSET=utf8 COLLATE=utf8_polish_ci;


Dodanie przykładowych danych do tabeli (przykład tabela Conntact).
